# Internal Development Portal

## Backstage

- Official: <https://backstage.io/>
- GitHub: <https://github.com/backstage/backstage>

CNCF Incubating open source platform from Spotify for building developer portals, with a software catalog, templates, and TechDocs.

## Port

- Official: <https://www.getport.io/>

SaaS internal developer portal focused on no-code catalog modeling, self-service actions, and scorecards for platform engineering.

## Kratix

- Official: <https://www.kratix.io/>
- GitHub: <https://github.com/syntasso/kratix>

Syntasso's open source framework for building platforms by composing Promises (capabilities) on top of Kubernetes.

## Humanitec

- Official: <https://humanitec.com/>

Platform Orchestrator that powers Internal Developer Platforms, dynamically generating configuration for app deployments across environments.

## Cortex

- Official: <https://www.cortex.io/>

Commercial internal developer portal centered on a service catalog, scorecards, and engineering productivity insights.

## Roadie

- Official: <https://roadie.io/>

Managed/SaaS Backstage offering, providing a hosted developer portal with plugins, integrations, and SSO out of the box.

## Configure8

- Official: <https://www.configure8.io/>

Internal developer portal that automates discovery of services and infrastructure from cloud and Kubernetes for catalog and scorecards.

## Rely

- Official: <https://rely.io/>

Internal developer portal platform that builds a graph of services, infrastructure, and ownership to power self-service workflows.

## CNOE (Cloud Native Operational Excellence)

- Official: <https://cnoe.io/>
- GitHub: <https://github.com/cnoe-io>

Open source framework and reference IDP from AWS and partners for building internal developer platforms using CNCF projects.

## Walrus

- Official: <https://seal-io.github.io/docs/>
- GitHub: <https://github.com/seal-io/walrus>

Seal's open source application management platform that simplifies multi-environment deployments on Kubernetes and other targets via Terraform.

## Kusion

- Official: <https://www.kusionstack.io/>
- GitHub: <https://github.com/KusionStack/kusion>

KusionStack's open source platform engineering tool using a dedicated configuration language (KCL) to model and deploy applications and infrastructure.
