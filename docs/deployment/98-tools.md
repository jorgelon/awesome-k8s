# Tools

## kube-linter

- Official: <https://docs.kubelinter.io/>
- GitHub: <https://github.com/stackrox/kube-linter>

StackRox tool that checks Kubernetes YAML and Helm charts for production-readiness and security best practices.

## kubeconform

- GitHub: <https://github.com/yannh/kubeconform>

Fast Kubernetes manifest validator using JSON schemas, drop-in replacement for kubeval.

## kubeval

- GitHub: <https://github.com/instrumenta/kubeval>

Original Kubernetes manifest schema validator (now mostly superseded by kubeconform).

## conftest

- Official: <https://www.conftest.dev/>
- GitHub: <https://github.com/open-policy-agent/conftest>

OPA-based tool for writing tests against structured configuration (YAML, JSON, HCL, Dockerfile).

## kube-score

- Official: <https://kube-score.com/>
- GitHub: <https://github.com/zegl/kube-score>

Static analysis of Kubernetes object definitions, recommending security and reliability improvements.

## Terraform

- Official: <https://www.terraform.io/>
- GitHub: <https://github.com/hashicorp/terraform>

HashiCorp's Infrastructure-as-Code CLI used heavily alongside Kubernetes for provisioning clusters and cloud resources.
