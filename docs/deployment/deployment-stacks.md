# Deployment Stacks

## Keptn

- Official: <https://keptn.sh/>
- GitHub: <https://github.com/keptn/lifecycle-toolkit>

CNCF Incubating cloud-native lifecycle toolkit for orchestrating deployments, with pre/post deployment tasks, SLO evaluation, and observability.

## Devtron

- Official: <https://devtron.ai/>
- GitHub: <https://github.com/devtron-labs/devtron>

Open source application delivery platform for Kubernetes integrating CI/CD, GitOps, security, and observability under one UI.

## Harness

- Official: <https://www.harness.io/>

Commercial software delivery platform covering CI, CD/GitOps, feature flags, cloud cost, chaos, and SEI — with a community Open Source edition.

## Spinnaker

- Official: <https://spinnaker.io/>
- GitHub: <https://github.com/spinnaker/spinnaker>

Multi-cloud continuous delivery platform originally from Netflix, supporting advanced deployment strategies across Kubernetes and clouds.

## Kubefirst

- Official: <https://kubefirst.io/>
- GitHub: <https://github.com/konstructio/kubefirst>

Open source instant GitOps platform that provisions opinionated Kubernetes clusters preloaded with Argo CD, Vault, Atlantis, and more.
