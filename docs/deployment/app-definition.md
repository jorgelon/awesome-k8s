# App Definition

## KubeVela

- Official: <https://kubevela.io/>
- GitHub: <https://github.com/kubevela/kubevela>

CNCF Incubating modern application delivery platform based on the Open Application Model (OAM), with multi-cluster and multi-environment support.

## Buildpacks

- Official: <https://buildpacks.io/>
- GitHub: <https://github.com/buildpacks>

CNCF Incubating Cloud Native Buildpacks transform application source code into OCI images without writing Dockerfiles.

## Kube-linter (analyze YAML)

- Official: <https://docs.kubelinter.io/>
- GitHub: <https://github.com/stackrox/kube-linter>

StackRox/Red Hat static analysis tool that checks Kubernetes YAML and Helm charts for production-readiness and security best practices.

## Kubeconform and JSON schemas

- GitHub (kubeconform): <https://github.com/yannh/kubeconform>
- Schemas GitHub: <https://github.com/yannh/kubernetes-json-schema>

Fast, configurable Kubernetes manifest validator that uses JSON schemas to validate resources against the cluster's API.

## Glasskube

- Official: <https://glasskube.dev/>
- GitHub: <https://github.com/glasskube/glasskube>

Next-generation package manager for Kubernetes with a UI/CLI, GitOps-friendly Packages, and dependency management.

## Kro (Kube Resource Orchestrator)

- Official: <https://kro.run/>
- GitHub: <https://github.com/kro-run/kro>

Open source project from AWS, GCP, and Azure for grouping Kubernetes resources into reusable APIs (ResourceGraphDefinitions).

## Carvel kapp

- Official: <https://carvel.dev/kapp/>
- GitHub: <https://github.com/carvel-dev/kapp>

Carvel tool that manages sets of Kubernetes resources as applications, with diff, converge, and label-based grouping.

## Crossplane

- Official: <https://www.crossplane.io/>
- GitHub: <https://github.com/crossplane/crossplane>

CNCF Graduated project that turns Kubernetes into a control plane for any cloud or service, exposed via Compositions and Composite Resources.
