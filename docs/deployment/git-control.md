# Git Control

## SemVer

- Official: <https://semver.org/>
- GitHub: <https://github.com/semver/semver>

Semantic Versioning specification — a versioning scheme of MAJOR.MINOR.PATCH that communicates intent of change and compatibility.

## GitVersion

- Official: <https://gitversion.net/>
- GitHub: <https://github.com/GitTools/GitVersion>

GitTools project that calculates a SemVer version number from your Git history and branching strategy, integrating with CI/CD pipelines.

## Renovate

- Official: <https://www.mend.io/renovate/>
- GitHub: <https://github.com/renovatebot/renovate>

Mend's open source dependency update bot, supporting many ecosystems (npm, Maven, Helm, Docker images, GitHub Actions, etc.) via pull requests.

## git-secrets

- GitHub: <https://github.com/awslabs/git-secrets>

AWS Labs Git hook that prevents committing AWS keys and other secret patterns to repositories, scanning commits, commit messages, and merges.

## Talisman

- GitHub: <https://github.com/thoughtworks/talisman>

Thoughtworks' Git pre-commit/pre-push hook that detects potential secrets and sensitive data in changes before they reach the repository.

## pre-commit

- Official: <https://pre-commit.com/>
- GitHub: <https://github.com/pre-commit/pre-commit>

Framework for managing and running multi-language Git pre-commit hooks, widely used to run linters, formatters, and secret scanners.

## Conventional Commits

- Official: <https://www.conventionalcommits.org/>
- GitHub: <https://github.com/conventional-commits/conventionalcommits.org>

Specification for adding human and machine readable meaning to commit messages, enabling automated changelogs and SemVer releases.

## Shell Tooling

### shfmt

- GitHub: <https://github.com/mvdan/sh>

Shell parser, formatter, and interpreter (POSIX, bash, mksh), used widely as a `gofmt` equivalent for shell scripts.

### shellcheck

- Official: <https://www.shellcheck.net/>
- GitHub: <https://github.com/koalaman/shellcheck>

Static analysis tool for shell scripts, catching common bugs and style issues.
