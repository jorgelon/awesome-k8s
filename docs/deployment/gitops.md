# GitOps

## Argo CD

- Official: <https://argo-cd.readthedocs.io/>
- GitHub: <https://github.com/argoproj/argo-cd>

CNCF Graduated declarative GitOps continuous delivery tool for Kubernetes, syncing application state from Git repositories.

## Flux CD

- Official: <https://fluxcd.io/>
- GitHub: <https://github.com/fluxcd/flux2>

CNCF Graduated GitOps toolkit for Kubernetes, providing modular controllers for sources, Helm, Kustomize, and notifications.

## Kapp Controller (Carvel)

- Official: <https://carvel.dev/kapp-controller/>
- GitHub: <https://github.com/carvel-dev/kapp-controller>

Carvel's Kubernetes controller for managing the lifecycle of applications declaratively, with packaging and templating integrations.

## Jenkins X

- Official: <https://jenkins-x.io/>
- GitHub: <https://github.com/jenkins-x/jx>

GitOps-based CI/CD platform for Kubernetes with automated pipelines, environments, and preview deployments using Tekton.

## Kluctl

- Official: <https://kluctl.io/>
- GitHub: <https://github.com/kluctl/kluctl>

CNCF Sandbox deployment tool for Kubernetes that combines templating and GitOps, with a strong focus on multi-environment deployments and diffs.

## PipeCD

- Official: <https://pipecd.dev/>
- GitHub: <https://github.com/pipe-cd/pipecd>

CNCF Sandbox continuous delivery system for declarative deployments to Kubernetes, Terraform, Lambda, Cloud Run, and ECS.

## Octopilot

- Official: <https://dailymotion-oss.github.io/octopilot/>
- GitHub: <https://github.com/dailymotion-oss/octopilot>

Dailymotion's open source CLI to automate updates across many Git repositories via pull requests — useful for GitOps version bumps and config rollouts.

## Fleet

- Official: <https://fleet.rancher.io/>
- GitHub: <https://github.com/rancher/fleet>

Rancher's GitOps engine designed for managing applications at scale across many Kubernetes clusters.
