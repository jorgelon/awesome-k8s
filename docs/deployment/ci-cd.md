# CI/CD

## Argo Workflows

- Official: <https://argoproj.github.io/workflows/>
- GitHub: <https://github.com/argoproj/argo-workflows>

CNCF Graduated container-native workflow engine for orchestrating parallel jobs on Kubernetes, widely used for CI and data pipelines.

## Tekton CD

- Official: <https://tekton.dev/>
- GitHub: <https://github.com/tektoncd/pipeline>

CDF Graduated Kubernetes-native CI/CD framework providing Pipelines, Tasks, Triggers, and Chains for supply chain security.

## Dagger

- Official: <https://dagger.io/>
- GitHub: <https://github.com/dagger/dagger>

Programmable CI/CD engine that runs pipelines as code in containers, with SDKs for Go, TypeScript, Python, and more.

## Jenkins

- Official: <https://www.jenkins.io/>
- GitHub: <https://github.com/jenkinsci/jenkins>

The de-facto open source automation server, with a vast plugin ecosystem and Kubernetes agents via the Kubernetes plugin.

## Screwdriver

- Official: <https://screwdriver.cd/>
- GitHub: <https://github.com/screwdriver-cd/screwdriver>

Open source build platform originally from Yahoo, designed for continuous delivery pipelines that scale.

## Couler

- GitHub: <https://github.com/couler-proj/couler>

Unified Python interface for defining and managing workflows across different engines (Argo, Tekton, Airflow).

## Kraken CI

- Official: <https://kraken.ci/>
- GitHub: <https://github.com/Kraken-CI/kraken>

Open source, modern CI/CD system focused on testing at scale, with parallel and distributed execution and rich result analysis.

## Werf

- Official: <https://werf.io/>
- GitHub: <https://github.com/werf/werf>

CNCF Sandbox CLI tool that integrates Git, Docker, Helm, and Kubernetes into a consistent CI/CD workflow for any CI system.

## GitLab Auto DevOps and CI/CD

- Official: <https://docs.gitlab.com/topics/autodevops/>

GitLab's built-in CI/CD with Auto DevOps templates that automatically build, test, and deploy applications to Kubernetes.

## Gitness

- Official: <https://gitness.com/>
- GitHub: <https://github.com/harness/gitness>

Harness' open source Git hosting and CI engine (now Harness Open Source), combining source control and pipelines in one binary.

## GitHub Actions

- Official: <https://github.com/features/actions>
- Docs: <https://docs.github.com/actions>

GitHub's integrated CI/CD platform driven by workflows in repositories, with self-hosted runners that can run on Kubernetes via Actions Runner Controller.
