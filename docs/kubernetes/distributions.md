# Distributions

## K3s ecosystem

- Official: <https://k3s.io/>
- GitHub: <https://github.com/k3s-io/k3s>

Lightweight, CNCF-certified Kubernetes distribution from Rancher/SUSE, packaged as a single binary and optimized for edge, IoT, CI, and ARM environments.

## Rancher

- Official: <https://www.rancher.com/>
- GitHub: <https://github.com/rancher/rancher>

Open source Kubernetes management platform from SUSE for provisioning, operating, and securing multiple clusters across on-prem and cloud environments.

## OpenShift

- Official: <https://www.redhat.com/en/technologies/cloud-computing/openshift>
- GitHub (OKD upstream): <https://github.com/openshift/okd>

Red Hat's enterprise Kubernetes distribution with integrated developer tooling, CI/CD, networking, and security. OKD is the upstream community edition.

## Karmada

- Official: <https://karmada.io/>
- GitHub: <https://github.com/karmada-io/karmada>

CNCF project providing multi-cluster and multi-cloud Kubernetes orchestration with a Kubernetes-native API for workload federation.

## Kubermatic

- Official: <https://www.kubermatic.com/products/kubermatic-kubernetes-platform/>
- GitHub: <https://github.com/kubermatic/kubermatic>

Open source Kubernetes management platform (KKP) for automating the lifecycle of thousands of clusters across multi-cloud, on-prem, and edge.

## k0s

- Official: <https://k0sproject.io/>
- GitHub: <https://github.com/k0sproject/k0s>

Mirantis' zero-friction Kubernetes distribution shipped as a single binary with no host OS dependencies, suitable for edge, bare metal, and cloud.

## vCluster

- Official: <https://www.vcluster.com/>
- GitHub: <https://github.com/loft-sh/vcluster>

Loft Labs' virtual Kubernetes clusters that run inside a namespace of a host cluster, providing strong isolation with minimal overhead.

## Amazon EKS Anywhere

- Official: <https://anywhere.eks.amazonaws.com/>
- GitHub: <https://github.com/aws/eks-anywhere>

AWS's on-premises Kubernetes distribution based on EKS Distro, deployable on bare metal, vSphere, Nutanix, CloudStack, and Snow.
