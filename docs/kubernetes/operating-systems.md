# Operating Systems for Containers

## Kairos

- Official: <https://kairos.io/>
- GitHub: <https://github.com/kairos-io/kairos>

Open source, immutable, container-based meta Linux distribution for building edge, cloud, and Kubernetes-native operating systems.

## Talos Linux

- Official: <https://www.talos.dev/>
- GitHub: <https://github.com/siderolabs/talos>

Sidero Labs' minimal, immutable, API-driven Linux distribution designed exclusively for running Kubernetes, with no shell or SSH.

## Flatcar Container Linux

- Official: <https://www.flatcar.org/>
- GitHub: <https://github.com/flatcar/Flatcar>

CNCF immutable Linux distribution for containers, a community continuation of CoreOS Container Linux maintained by Microsoft.

## VMware Photon OS

- Official: <https://vmware.github.io/photon/>
- GitHub: <https://github.com/vmware/photon>

VMware's minimal Linux container host operating system optimized for vSphere and cloud-native workloads.

## Kubermatic Operating System Manager (OSM)

- Official: <https://docs.kubermatic.com/kubermatic/main/architecture/concept/kkp-concepts/operating-system-manager/>
- GitHub: <https://github.com/kubermatic/operating-system-manager>

Kubermatic component that manages the lifecycle of node operating systems via user-data templates, supporting multiple OS flavors in KKP/KubeOne.
