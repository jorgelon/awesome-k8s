# Baremetal and Infra

## Harvester

- Official: <https://harvesterhci.io/>
- GitHub: <https://github.com/harvester/harvester>

SUSE/Rancher's open source hyperconverged infrastructure (HCI) solution built on Kubernetes, KubeVirt, and Longhorn for running VMs and containers on bare metal.

## MAAS

- Official: <https://maas.io/>
- GitHub: <https://github.com/canonical/maas>

Canonical's Metal as a Service: provisions physical servers like cloud instances with PXE, DHCP, and a REST API for bare-metal lifecycle management.

## Metal3

- Official: <https://metal3.io/>
- GitHub: <https://github.com/metal3-io/metal3-docs>

Bare-metal host provisioning project that integrates with Cluster API to manage physical servers using Kubernetes-native APIs and Ironic.

## Tinkerbell

- Official: <https://tinkerbell.org/>
- GitHub: <https://github.com/tinkerbell/tink>

CNCF bare-metal provisioning engine (originally from Equinix Metal) providing DHCP, iPXE, OS installation, and workflow execution on physical hardware.

## KubeVirt

- Official: <https://kubevirt.io/>
- GitHub: <https://github.com/kubevirt/kubevirt>

CNCF project that runs virtual machines as first-class Kubernetes workloads alongside containers, using KVM under the hood.

## Claudie

- Official: <https://docs.claudie.io/>
- GitHub: <https://github.com/berops/claudie>

Berops' open source platform for building multi-cloud and hybrid Kubernetes clusters that span multiple providers and on-prem environments.
