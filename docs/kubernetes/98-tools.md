# Tools

## kubectx

- GitHub: <https://github.com/ahmetb/kubectx>

Fast switch between Kubernetes clusters (contexts) from the command line.

## kubens

- GitHub: <https://github.com/ahmetb/kubectx>

Companion to kubectx for switching between namespaces in the current context.

## kubie

- GitHub: <https://github.com/sbstp/kubie>

Alternative to kubectx/kubens that isolates context and namespace per shell, with a colorful prompt.

## kube-ps1

- GitHub: <https://github.com/jonmosco/kube-ps1>

Bash/Zsh prompt customization showing the current Kubernetes context and namespace.

## Stern

- GitHub: <https://github.com/stern/stern>

Multi-pod, multi-container log tailer for Kubernetes, with regex filtering and coloring.

## krew

- Official: <https://krew.sigs.k8s.io/>
- GitHub: <https://github.com/kubernetes-sigs/krew>

Official kubectl plugin manager from Kubernetes SIG CLI.

## asdf

- Official: <https://asdf-vm.com/>
- GitHub: <https://github.com/asdf-vm/asdf>

Extendable version manager for many runtimes and CLI tools (kubectl, helm, terraform, etc.) via plugins.

## Butane

- Official: <https://coreos.github.io/butane/>
- GitHub: <https://github.com/coreos/butane>

CoreOS tool that translates human-readable Butane Config into Ignition for provisioning Flatcar/Fedora CoreOS nodes.
