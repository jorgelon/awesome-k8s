# Managed Kubernetes

## Amazon EKS

- Official: <https://aws.amazon.com/eks/>

Amazon Elastic Kubernetes Service is AWS's managed Kubernetes offering, handling control plane provisioning, upgrades, and integration with the AWS ecosystem.

## Google Kubernetes Engine (GKE)

- Official: <https://cloud.google.com/kubernetes-engine>

Google Cloud's managed Kubernetes service, built by the same team that originated Kubernetes, with autopilot and standard modes.

## Azure Kubernetes Service (AKS)

- Official: <https://azure.microsoft.com/products/kubernetes-service>

Microsoft Azure's managed Kubernetes service with deep integration into Entra ID, Azure Monitor, and the broader Azure platform.

## Civo

- Official: <https://www.civo.com/kubernetes>

Civo offers a Kubernetes service powered by K3s, focused on fast cluster launch times and developer-friendly pricing.

## Digital Ocean Managed Kubernetes

- Official: <https://www.digitalocean.com/products/kubernetes>

DigitalOcean's managed Kubernetes offering with a simple UX, integrated load balancers and block storage, and predictable pricing.

## Vultr Kubernetes Engine (VKE)

- Official: <https://www.vultr.com/kubernetes/>

Vultr's managed Kubernetes service available across its global cloud regions, with a free control plane option.

## Scaleway Kapsule

- Official: <https://www.scaleway.com/en/kubernetes-kapsule/>

Scaleway's managed Kubernetes offering hosted in European data centers, with multi-AZ control planes (Kosmos for hybrid pools).

## Linode Kubernetes Engine (LKE)

- Official: <https://www.linode.com/products/kubernetes/>

Akamai/Linode's managed Kubernetes service with a free control plane and integration with Linode block storage and load balancers.

## UpCloud Managed Kubernetes

- Official: <https://upcloud.com/products/managed-kubernetes>

UpCloud's managed Kubernetes service emphasizing performance and European data residency.

## OVHcloud Managed Kubernetes

- Official: <https://www.ovhcloud.com/en/public-cloud/kubernetes/>

OVHcloud's managed Kubernetes service with a free control plane, hosted across OVH's European and global regions.
