# Observability

## OpenTelemetry

- Official: <https://opentelemetry.io/>
- GitHub: <https://github.com/open-telemetry>

CNCF Incubating set of APIs, SDKs, and tools for generating, collecting, and exporting telemetry (traces, metrics, logs) as a vendor-neutral standard.

## Elasticsearch

- Official: <https://www.elastic.co/elasticsearch>
- GitHub: <https://github.com/elastic/elasticsearch>

Elastic's distributed search and analytics engine, widely used as a backend for logs, traces, and observability data via the Elastic Stack.

## OpenSearch

- Official: <https://opensearch.org/>
- GitHub: <https://github.com/opensearch-project/OpenSearch>

Linux Foundation community-driven fork of Elasticsearch and Kibana, providing search, observability, security analytics, and ML capabilities.
