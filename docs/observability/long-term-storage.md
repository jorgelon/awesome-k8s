# Long Term Storage

## Cortex

- Official: <https://cortexmetrics.io/>
- GitHub: <https://github.com/cortexproject/cortex>

CNCF Incubating horizontally scalable, multi-tenant, long-term storage for Prometheus metrics, originally built by Weaveworks/Grafana.

## Thanos

- Official: <https://thanos.io/>
- GitHub: <https://github.com/thanos-io/thanos>

CNCF Incubating highly available Prometheus setup with long-term storage, global query view, downsampling, and deduplication via object storage.

## VictoriaMetrics

- Official: <https://victoriametrics.com/>
- GitHub: <https://github.com/VictoriaMetrics/VictoriaMetrics>

Fast, cost-effective, scalable time series database and monitoring solution, Prometheus-compatible with a vertical and clustered version.

## Grafana Mimir

- Official: <https://grafana.com/oss/mimir/>
- GitHub: <https://github.com/grafana/mimir>

Grafana's open source horizontally scalable, multi-tenant TSDB for Prometheus and OpenTelemetry metrics, a Cortex evolution from Grafana Labs.

## GreptimeDB

- Official: <https://greptime.com/>
- GitHub: <https://github.com/GreptimeTeam/greptimedb>

Open source cloud-native time series database with SQL and PromQL support, designed for unified metrics, logs, and events.
