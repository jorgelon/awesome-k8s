- Official: <https://superset.apache.org/>

Apache modern data exploration and visualization platform, with rich SQL Lab, dashboards, and a wide range of database connectors.

## Graphite

- Official: <https://graphiteapp.org/>
- GitHub: <https://github.com/graphite-project/graphite-web>

Long-running open source time-series database and web app for storing and visualizing numeric metrics, with the carbon ingestion stack.
