# Tracing

## Jaeger

- Official: <https://www.jaegertracing.io/>
- GitHub: <https://github.com/jaegertracing/jaeger>

CNCF Graduated end-to-end distributed tracing system originally from Uber, supporting OpenTelemetry and many storage backends.

## Grafana Tempo

- Official: <https://grafana.com/oss/tempo/>
- GitHub: <https://github.com/grafana/tempo>

Grafana's open source, high-scale, easy-to-operate distributed tracing backend with object storage, integrated with Grafana, Loki, and Mimir.

## Grafana Beyla

- Official: <https://grafana.com/oss/beyla-ebpf/>
- GitHub: <https://github.com/grafana/beyla>

Grafana's eBPF-based zero-code auto-instrumentation that generates RED metrics and traces for applications without modifying their code.

## Apache SkyWalking

- Official: <https://skywalking.apache.org/>
- GitHub: <https://github.com/apache/skywalking>

Apache distributed APM and observability platform with tracing, metrics, logs, eBPF profiling, and service mesh integrations.

## Zipkin

- Official: <https://zipkin.io/>
- GitHub: <https://github.com/openzipkin/zipkin>

One of the original distributed tracing systems (originally from Twitter), with a simple, single-binary deployment model.

## Grafana Pyroscope

- GitHub: <https://github.com/grafana/pyroscope>

Grafana's open source continuous profiling backend (formerly Pyroscope) that stores and queries CPU/memory profiles alongside metrics, logs, and traces.

## Elastic APM

- Official: <https://www.elastic.co/observability/application-performance-monitoring>
- GitHub: <https://github.com/elastic/apm-server>

Elastic's application performance monitoring solution built on Elasticsearch, with agents for many languages and OpenTelemetry support.

## bcc

- GitHub: <https://github.com/iovisor/bcc>

IO Visor BPF Compiler Collection — a toolkit and Python/Lua front-ends for writing efficient eBPF-based kernel tracing and analysis tools.

## bpftrace

- Official: <https://bpftrace.org/>
- GitHub: <https://github.com/bpftrace/bpftrace>

High-level tracing language for Linux eBPF, inspired by awk, DTrace, and SystemTap, for one-liners and short scripts.

## OpenTelemetry eBPF Profiler

- GitHub: <https://github.com/open-telemetry/opentelemetry-ebpf-profiler>

Whole-system, cross-language continuous profiler for Linux donated by Elastic (formerly Universal Profiling) to the OpenTelemetry project.

## Coroot

- Official: <https://coroot.com/>
- GitHub: <https://github.com/coroot/coroot>

Open source eBPF-based observability platform that maps service topology and provides RED metrics, traces, and SLO insights with low overhead.
