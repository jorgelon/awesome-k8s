# Logging

## Fluentd

- Official: <https://www.fluentd.org/>
- GitHub: <https://github.com/fluent/fluentd>

CNCF Graduated open source data collector for unified logging, with a rich plugin ecosystem for sources, parsers, filters, and outputs.

## Grafana Loki and Promtail

- Loki official: <https://grafana.com/oss/loki/>
- Loki GitHub: <https://github.com/grafana/loki>
- Promtail GitHub: <https://github.com/grafana/loki/tree/main/clients/pkg/promtail>

Grafana's horizontally scalable log aggregation system inspired by Prometheus, with Promtail (and now Grafana Alloy) as the shipping agent.

## Logstash

- Official: <https://www.elastic.co/logstash>
- GitHub: <https://github.com/elastic/logstash>

Elastic's server-side data processing pipeline that ingests, transforms, and ships data to Elasticsearch or other backends.

## Fluent Bit

- Official: <https://fluentbit.io/>
- GitHub: <https://github.com/fluent/fluent-bit>

CNCF Graduated lightweight log and metric processor and forwarder, the recommended Fluent project for Kubernetes node-level collection.
