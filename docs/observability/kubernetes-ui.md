# Kubernetes UI

## k9s

- Official: <https://k9scli.io/>
- GitHub: <https://github.com/derailed/k9s>

Terminal UI to interact with Kubernetes clusters in real time, with resource navigation, log streaming, and powerful keyboard-driven workflows.

## Kui

- Official: <https://kui.tools/>
- GitHub: <https://github.com/kubernetes-sigs/kui>

IBM/SIG project that turns CLI commands into visual responses, blending a terminal with graphical tables, charts, and editors.

## Skooner

- GitHub: <https://github.com/skooner-k8s/skooner>

Open source Kubernetes dashboard (formerly k8dash) with a simple, fast UI for managing cluster resources via a single page web app.

## Headlamp

- Official: <https://headlamp.dev/>
- GitHub: <https://github.com/headlamp-k8s/headlamp>

CNCF Sandbox extensible Kubernetes web UI and desktop app from Microsoft/Kinvolk, with plugin support and multi-cluster management.

## Kubevious

- Official: <https://kubevious.io/>
- GitHub: <https://github.com/kubevious/kubevious>

Application-centric Kubernetes dashboard with validation rules, configuration insights, and time-machine views of cluster state.

## Lens and OpenLens

- Lens official: <https://k8slens.dev/>
- Lens GitHub: <https://github.com/lensapp/lens>
- OpenLens GitHub: <https://github.com/MuhammedKalkan/OpenLens>

Mirantis' Lens is a popular Kubernetes IDE; OpenLens is the open source build of the core, packaged by the community.

## k8Studio

- Official: <https://k8studio.io/>

Commercial Kubernetes IDE/desktop client with multi-cluster management, visualization, and built-in tooling for day-to-day operations.

## Forecastle

- GitHub: <https://github.com/stakater/Forecastle>

Stakater's open source control panel that auto-discovers and launches applications running on Kubernetes, grouped by namespace.

## Visually Presented Kubernetes (VPK)

- Official: <https://k8svisual.github.io/vpk-docs/>
- GitHub: <https://github.com/k8svisual/vpk>

Open source tool that visually presents Kubernetes resources and their relationships from offline YAML, helping understand and document cluster state.

## Radar

- Official: <https://radarhq.io/>
- GitHub: <https://github.com/skyhook-io/radar>

Apache 2.0 Kubernetes UI from Skyhook that visualizes cluster topology, events, Helm releases, and GitOps state in real time, with fleet-wide search and extended event retention from a single binary deployable locally or in-cluster.
