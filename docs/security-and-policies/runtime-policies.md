# Runtime Policies

## Falco

- Official: <https://falco.org/>
- GitHub: <https://github.com/falcosecurity/falco>

CNCF Graduated runtime security tool that detects anomalous behavior in containers, hosts, and Kubernetes using eBPF/kernel module probes.

## KubeArmor

- Official: <https://kubearmor.io/>
- GitHub: <https://github.com/kubearmor/KubeArmor>

CNCF Sandbox runtime security enforcement for containers, pods, VMs, and hosts using LSMs (AppArmor, SELinux, BPF-LSM).

## Tetragon

- Official: <https://tetragon.io/>
- GitHub: <https://github.com/cilium/tetragon>

Cilium's eBPF-based security observability and runtime enforcement platform, providing deep visibility and policy at the kernel level.

## Tracee

- Official: <https://aquasecurity.github.io/tracee/>
- GitHub: <https://github.com/aquasecurity/tracee>

Aqua Security's eBPF-based runtime security and forensics tool that detects suspicious behavioral patterns in Linux/Kubernetes workloads.

## BitNinja

- Official: <https://bitninja.com/>

Commercial server security platform providing runtime protection (WAF, malware scanning, intrusion detection) for Linux servers and container hosts.

## Inspektor Gadget

- Official: <https://inspektor-gadget.io/>
- GitHub: <https://github.com/inspektor-gadget/inspektor-gadget>

CNCF Sandbox collection of eBPF-based tools (gadgets) to inspect, trace, and debug applications and the Kubernetes node.
