# Identity

## Dex

- Official: <https://dexidp.io/>
- GitHub: <https://github.com/dexidp/dex>

CNCF Graduated OpenID Connect identity provider that federates authentication from upstream IdPs (LDAP, SAML, GitHub, Google, etc.).

## Keycloak

- Official: <https://www.keycloak.org/>
- GitHub: <https://github.com/keycloak/keycloak>

CNCF Incubating open source identity and access management, providing SSO, OIDC, SAML, social login, and a Kubernetes operator.

## SPIFFE and SPIRE

- Official: <https://spiffe.io/>
- GitHub (SPIFFE): <https://github.com/spiffe/spiffe>
- GitHub (SPIRE): <https://github.com/spiffe/spire>

CNCF Graduated standard (SPIFFE) and runtime (SPIRE) for issuing cryptographic workload identities in heterogeneous environments.

## cert-manager

- Official: <https://cert-manager.io/>
- GitHub: <https://github.com/cert-manager/cert-manager>

CNCF Graduated Kubernetes certificate management controller that automates issuance and renewal from ACME, Vault, Venafi, and other CAs.

## Twingate

- Official: <https://www.twingate.com/>
- GitHub: <https://github.com/Twingate>

Commercial zero-trust network access platform with a Kubernetes operator to securely expose private services without VPNs.

## OAuth2 Proxy

- GitHub: <https://github.com/oauth2-proxy/oauth2-proxy>

Reverse proxy and static file server that provides authentication via OAuth2/OIDC providers, commonly used in front of Kubernetes apps and Ingress.
