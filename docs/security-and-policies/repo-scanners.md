# Repo Scanners

## TruffleHog

- Official: <https://trufflesecurity.com/trufflehog>
- GitHub: <https://github.com/trufflesecurity/trufflehog>

Truffle Security's open source tool that scans repositories, files, and live cloud assets for secrets and verifies them against the source provider.

## detect-secrets

- GitHub: <https://github.com/Yelp/detect-secrets>

Yelp's enterprise-friendly tool for detecting and preventing secrets in code, with a baseline workflow to manage existing findings.

## Gitleaks

- Official: <https://gitleaks.io/>
- GitHub: <https://github.com/gitleaks/gitleaks>

Fast, configurable SAST tool for detecting hard-coded secrets in Git repositories, history, and CI pipelines.

## GitGuardian

- Official: <https://www.gitguardian.com/>
- GitHub: <https://github.com/GitGuardian/ggshield>

Commercial secrets detection platform with the open source ggshield CLI for local and CI secret, IaC, and SCA scanning.

## Talisman

- GitHub: <https://github.com/thoughtworks/talisman>

Thoughtworks' Git pre-commit/pre-push hook that detects potential secrets and sensitive data in changes before they reach the repository.

## Trivy

- Official: <https://trivy.dev/>
- GitHub: <https://github.com/aquasecurity/trivy>

Aqua Security's all-in-one scanner for vulnerabilities, misconfigurations, secrets, SBOMs, and licenses across containers, IaC, and repositories.

## Semgrep

- Official: <https://semgrep.dev/>
- GitHub: <https://github.com/semgrep/semgrep>

Lightweight static analysis tool with thousands of community rules across many languages for security, correctness, and policy.
