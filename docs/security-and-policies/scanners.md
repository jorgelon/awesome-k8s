# Scanners

## Trivy

- Official: <https://trivy.dev/>
- GitHub: <https://github.com/aquasecurity/trivy>

Aqua Security's all-in-one scanner for vulnerabilities, misconfigurations, secrets, SBOMs, and licenses across containers, IaC, Kubernetes, and repositories.

## OpenClarity

- Official: <https://openclarity.io/>
- GitHub: <https://github.com/openclarity/openclarity>

Open source platform (originally KubeClarity) for SBOM generation and vulnerability, malware, secret, misconfiguration, and IaC scanning.

## Kubeshark

- Official: <https://www.kubeshark.co/>
- GitHub: <https://github.com/kubeshark/kubeshark>

API traffic viewer for Kubernetes that captures and inspects all in-cluster traffic and APIs in real time, useful for security and debugging.

## OWASP ZAP

- Official: <https://www.zaproxy.org/>
- GitHub: <https://github.com/zaproxy/zaproxy>

OWASP's open source web application security scanner (Zed Attack Proxy) for finding vulnerabilities via active and passive scans.

## Copacetic (copa)

- Official: <https://project-copacetic.github.io/copacetic/>
- GitHub: <https://github.com/project-copacetic/copacetic>

CNCF Sandbox project that directly patches OS-level vulnerabilities in container images using Trivy/scanner reports, without rebuilding from source.

## Kubescape

- Official: <https://kubescape.io/>
- GitHub: <https://github.com/kubescape/kubescape>

CNCF Incubating Kubernetes security platform that scans clusters, manifests, and Helm charts against frameworks like NSA-CISA and MITRE ATT&CK.

## k8s-cleaner

- GitHub: <https://github.com/gianlucam76/k8s-cleaner>

Kubernetes controller that identifies unused or stale resources and optionally cleans them up based on configurable policies and schedules.

## kube-hunter

- Official: <https://aquasecurity.github.io/kube-hunter/>
- GitHub: <https://github.com/aquasecurity/kube-hunter>

Aqua Security tool that hunts for security weaknesses in Kubernetes clusters from an attacker's perspective, externally or in-cluster.

## Clair

- Official: <https://quay.github.io/clair/>
- GitHub: <https://github.com/quay/clair>

Red Hat/Quay's open source vulnerability scanner for OCI containers, used by Quay and other registries to inspect images for known CVEs.

## Notary Project

- Official: <https://notaryproject.dev/>
- GitHub: <https://github.com/notaryproject>

CNCF Incubating project (Notary v2 / Notation) for signing and verifying OCI artifacts and container images, with Kubernetes admission integrations.

## Kubenomicon

- Official: <https://kubenomicon.com/>

Open knowledge base / threat matrix that maps real-world Kubernetes attack techniques to detection and mitigation guidance.

## Checkov

- Official: <https://www.checkov.io/>
- GitHub: <https://github.com/bridgecrewio/checkov>

Bridgecrew/Prisma Cloud static analysis tool for IaC (Terraform, CloudFormation, Kubernetes, Helm, Dockerfile, ARM) and policy as code.
