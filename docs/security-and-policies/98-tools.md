# Tools

## kubescape

- Official: <https://kubescape.io/>
- GitHub: <https://github.com/kubescape/kubescape>

CNCF Incubating Kubernetes security platform that scans clusters, manifests, and Helm charts against frameworks like NSA-CISA.

## kubeaudit

- GitHub: <https://github.com/Shopify/kubeaudit>

Shopify's CLI that audits Kubernetes clusters against common security controls, with autofix support for many findings.

## rakkess

- GitHub: <https://github.com/corneliusweig/rakkess>

`kubectl access-matrix` plugin that shows which subjects can perform which verbs on which resources.
