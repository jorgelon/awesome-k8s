# RBAC

## rbac-lookup

- GitHub: <https://github.com/FairwindsOps/rbac-lookup>

Fairwinds' CLI that lists Kubernetes role bindings filtered by user, service account, or group across the cluster.

## RBAC Manager

- GitHub: <https://github.com/FairwindsOps/rbac-manager>

Fairwinds' operator that provides higher-level abstractions for managing RBAC at scale via a single RBACDefinition CRD per subject.

## Permission Manager

- GitHub: <https://github.com/sighupio/permission-manager>

SIGHUP's web UI for creating, editing, and deleting users and RBAC permissions in a Kubernetes cluster.

## Rakkess

- GitHub: <https://github.com/corneliusweig/rakkess>

`kubectl access-matrix` plugin that shows an access matrix of which subjects can perform which verbs on which resources.
