# Secret Storage

## OpenBao

- Official: <https://openbao.org/>
- GitHub: <https://github.com/openbao/openbao>

Linux Foundation open source fork of HashiCorp Vault, providing secrets management, encryption-as-a-service, and identity-based access.

## HashiCorp Vault

- Official: <https://www.vaultproject.io/>
- GitHub: <https://github.com/hashicorp/vault>

Secrets management platform with dynamic credentials, PKI, encryption-as-a-service, and tight Kubernetes integration via the Vault Operator and Agent.

## Infisical

- Official: <https://infisical.com/>
- GitHub: <https://github.com/Infisical/infisical>

Open source secrets management platform with secret scanning, Kubernetes operator, and SDKs across many languages.

## Passbolt

- Official: <https://www.passbolt.com/>
- GitHub: <https://github.com/passbolt/passbolt_api>

Open source password manager designed for teams, self-hostable with role-based access and end-to-end encryption.

## Sealed Secrets

- GitHub: <https://github.com/bitnami-labs/sealed-secrets>

Bitnami Labs controller that encrypts Kubernetes Secrets into SealedSecret CRDs safe to store in Git, decrypted only inside the cluster.

## gopass

- Official: <https://www.gopass.pw/>
- GitHub: <https://github.com/gopasspw/gopass>

Team-oriented password manager that builds on `pass`, using GPG and Git for storage, with templates, secrets sharing, and many backends.

## Novops

- Official: <https://novops.dev/>
- GitHub: <https://github.com/PierreBeucher/novops>

Cross-platform secret aggregator that loads secrets from multiple sources (Vault, AWS, GCP, Azure, BitWarden, etc.) into environment variables or files for local dev and CI.

## Teller

- Official: <https://tellerops.com/>
- GitHub: <https://github.com/tellerops/teller>

Universal CLI/SDK for fetching, injecting, and scanning secrets across many providers (Vault, AWS, GCP, Doppler, 1Password, etc.) for local dev and CI.

## CyberArk Conjur

- Official: <https://www.conjur.org/>
- GitHub: <https://github.com/cyberark/conjur>

CyberArk's open source secrets management for containers and DevOps, providing RBAC-based access to secrets and Kubernetes authn integrations.
