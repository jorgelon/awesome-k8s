# FinOps and CO2

## Kubecost

- Official: <https://www.kubecost.com/>
- GitHub: <https://github.com/kubecost/cost-analyzer-helm-chart>

IBM/Kubecost's Kubernetes cost monitoring and optimization platform, providing real-time cost allocation, savings recommendations, and budgeting.

## OpenCost

- Official: <https://www.opencost.io/>
- GitHub: <https://github.com/opencost/opencost>

CNCF Incubating open source project (based on Kubecost) for measuring and allocating Kubernetes cluster costs across workloads.

## Crane

- Official: <https://gocrane.io/>
- GitHub: <https://github.com/gocrane/crane>

CNCF Sandbox FinOps platform from Tencent that recommends and applies resource right-sizing, autoscaling, and idle node optimizations on Kubernetes.

## Infracost

- Official: <https://www.infracost.io/>
- GitHub: <https://github.com/infracost/infracost>

Open source tool that shows cloud cost estimates for Terraform/OpenTofu changes directly in pull requests.

## amazon-ec2-instance-selector

- GitHub: <https://github.com/aws/amazon-ec2-instance-selector>

AWS open source CLI/library that filters and ranks EC2 instance types by resource and pricing criteria, useful for Karpenter/Cluster Autoscaler sizing.

## Kube-green

- Official: <https://kube-green.dev/>
- GitHub: <https://github.com/kube-green/kube-green>

Kubernetes operator that shuts down non-production workloads on a schedule to reduce CO2 emissions and cloud cost.

## Kepler

- Official: <https://sustainable-computing.io/>
- GitHub: <https://github.com/sustainable-computing-io/kepler>

CNCF Sandbox Kubernetes-based Efficient Power Level Exporter that uses eBPF to estimate energy consumption of pods and exports it as Prometheus metrics.

## Cloud Cost Exporter

- GitHub: <https://github.com/grafana/cloudcost-exporter>

Grafana's Prometheus exporter that exposes cloud provider (AWS, GCP, Azure) pricing as metrics, enabling cost dashboards and alerting in Grafana.

## CAST AI

- Official: <https://cast.ai/>

Commercial Kubernetes automation platform providing cost monitoring, autoscaling, spot instance management, and security posture across cloud providers.

## Finala

- GitHub: <https://github.com/similarweb/finala>

SimilarWeb's open source resource and cost cloud analyzer that detects waste across AWS services and provides actionable savings recommendations.
