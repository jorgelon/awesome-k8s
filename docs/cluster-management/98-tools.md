# Tools

## k8sgpt

- Official: <https://k8sgpt.ai/>
- GitHub: <https://github.com/k8sgpt-ai/k8sgpt>

CNCF Sandbox tool that scans Kubernetes clusters and diagnoses issues in plain English using LLMs.

## kopylot

- GitHub: <https://github.com/avsthiago/kopylot>

AI-powered kubectl assistant that suggests commands, audits resources, and explains errors.

## Popeye

- Official: <https://popeyecli.io/>
- GitHub: <https://github.com/derailed/popeye>

Cluster sanitizer that scans for misconfigurations, dead resources, and best-practice violations.

## Ketall

- GitHub: <https://github.com/corneliusweig/ketall>

`kubectl get-all` plugin that lists every resource (including CRDs) in a Kubernetes cluster.

## k8s-cleaner

- GitHub: <https://github.com/gianlucam76/k8s-cleaner>

Controller and CLI for identifying and removing stale/unused Kubernetes resources via configurable policies.

## kubent (kube-no-trouble)

- GitHub: <https://github.com/doitintl/kube-no-trouble>

Detects deprecated Kubernetes API usage in live clusters and Helm charts/manifests before an upgrade.

## Silver Surfer

- GitHub: <https://github.com/devtron-labs/silver-surfer>

Devtron tool that checks Kubernetes manifests for deprecations across versions and provides migration guidance.

## Pluto

- GitHub: <https://github.com/FairwindsOps/pluto>

Fairwinds CLI that detects deprecated and removed Kubernetes API versions in code, Helm releases, and manifests.

## kubepug

- Official: <https://kubepug.xyz/>
- GitHub: <https://github.com/kubepug/kubepug>

Kubernetes Pre-Upgrade Checker that scans clusters and manifests for deprecated APIs.

## kubepug web

- GitHub: <https://github.com/kubepug/kubepug-web>

Web UI for kubepug, providing an interactive deprecation report for Kubernetes resources.

## Nova

- GitHub: <https://github.com/FairwindsOps/nova>

Fairwinds CLI that finds outdated or deprecated Helm charts and container images running in your cluster.
