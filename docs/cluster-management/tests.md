# Tests

## KUTTL

- Official: <https://kuttl.dev/>
- GitHub: <https://github.com/kudobuilder/kuttl>

Kubernetes Test Tool: declarative test framework for testing operators and other Kubernetes resources using YAML test cases.

## Kuberhealthy

- Official: <https://kuberhealthy.com/>
- GitHub: <https://github.com/kuberhealthy/kuberhealthy>

CNCF Sandbox synthetic monitoring tool that runs checks as Kubernetes Jobs and exposes results as Prometheus metrics.

## K8sGPT

- Official: <https://k8sgpt.ai/>
- GitHub: <https://github.com/k8sgpt-ai/k8sgpt>

CNCF Sandbox tool that scans Kubernetes clusters and diagnoses issues in plain English, using LLMs to enrich findings.

## kube-bench

- GitHub: <https://github.com/aquasecurity/kube-bench>

Aqua Security tool that checks whether Kubernetes is deployed securely by running the CIS Kubernetes Benchmark.

## Testkube

- Official: <https://testkube.io/>
- GitHub: <https://github.com/kubeshop/testkube>

Kubernetes-native testing framework that runs and orchestrates many testing tools (k6, Cypress, Postman, JMeter, etc.) as CRDs.

## Grafana k6

- Official: <https://k6.io/>
- GitHub: <https://github.com/grafana/k6>

Grafana's modern open source load testing tool with a developer-friendly JavaScript scripting API and a Kubernetes operator.

## DDosify

- Official: <https://ddosify.com/>
- GitHub: <https://github.com/ddosify/ddosify>

Open source performance testing platform and load engine, with a Kubernetes-native control plane for distributed load tests.

## Keploy

- Official: <https://keploy.io/>
- GitHub: <https://github.com/keploy/keploy>

Open source tool that auto-generates test cases and mocks from real API traffic, enabling fast regression testing of microservices.

## Kyverno Chainsaw

- Official: <https://kyverno.github.io/chainsaw/>
- GitHub: <https://github.com/kyverno/chainsaw>

Kyverno's declarative end-to-end testing tool for Kubernetes operators and controllers, evolving from KUTTL with extended assertions.

## Kube-burner

- Official: <https://kube-burner.github.io/kube-burner/>
- GitHub: <https://github.com/kube-burner/kube-burner>

CNCF Sandbox tool for stress-testing and benchmarking Kubernetes clusters by creating large numbers of objects and collecting metrics.
