# Scalers

## KEDA

- Official: <https://keda.sh/>
- GitHub: <https://github.com/kedacore/keda>

CNCF Graduated Kubernetes Event-Driven Autoscaler that scales workloads based on external event sources (queues, streams, databases, cron, etc.).

## Kube-green

- Official: <https://kube-green.dev/>
- GitHub: <https://github.com/kube-green/kube-green>

Open source Kubernetes operator that automatically shuts down non-production workloads on a schedule to reduce CO2 emissions and cost.

## Snorlax

- GitHub: <https://github.com/moonbeam-nyc/snorlax>

Kubernetes operator that puts Deployments to sleep on a schedule (and wakes them on demand) to save resources in dev/staging environments.

## Horizontal Pod Autoscaler (HPA)

- Official: <https://kubernetes.io/docs/tasks/run-application/horizontal-pod-autoscale/>

Built-in Kubernetes controller that automatically scales the number of pods in a workload based on observed CPU, memory, or custom metrics.

## Vertical Pod Autoscaler (VPA)

- GitHub: <https://github.com/kubernetes/autoscaler/tree/master/vertical-pod-autoscaler>

Kubernetes SIG component that automatically sets/recommends container resource requests and limits based on historical usage.
