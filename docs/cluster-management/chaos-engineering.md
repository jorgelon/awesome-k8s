# Chaos Engineering

## Chaos Mesh

- Official: <https://chaos-mesh.org/>
- GitHub: <https://github.com/chaos-mesh/chaos-mesh>

CNCF Incubating chaos engineering platform for Kubernetes, providing many fault injection types (pod, network, IO, time, kernel) as CRDs.

## LitmusChaos

- Official: <https://litmuschaos.io/>
- GitHub: <https://github.com/litmuschaos/litmus>

CNCF Incubating end-to-end chaos engineering framework for Kubernetes with a hub of experiments and a GitOps-friendly control plane.

## PowerfulSeal

- GitHub: <https://github.com/powerfulseal/powerfulseal>

Open source chaos engineering tool for Kubernetes that kills targeted pods and nodes interactively, autonomously, or based on policy.

## ChaosBlade

- Official: <https://chaosblade.io/>
- GitHub: <https://github.com/chaosblade-io/chaosblade>

CNCF Sandbox chaos engineering tool from Alibaba covering OS-, JVM-, container-, and Kubernetes-level fault injection scenarios.
