# Multicluster

## Liqo

- Official: <https://liqo.io/>
- GitHub: <https://github.com/liqotech/liqo>

Open source project enabling dynamic, peer-to-peer Kubernetes cluster federation with resource sharing and seamless multi-cluster pod offloading.

## Skupper

- Official: <https://skupper.io/>
- GitHub: <https://github.com/skupperproject/skupper>

Red Hat-led Layer 7 service interconnect that creates virtual application networks across clusters and clouds without VPNs.

## Submariner

- Official: <https://submariner.io/>
- GitHub: <https://github.com/submariner-io/submariner>

CNCF Sandbox project providing cross-cluster L3 connectivity and service discovery between Kubernetes clusters across on-prem and cloud.

## Cilium Cluster Mesh

- Official: <https://docs.cilium.io/en/stable/network/clustermesh/>
- GitHub: <https://github.com/cilium/cilium>

Cilium feature that connects multiple Kubernetes clusters at the network layer, with global services, identity-aware policies, and shared Hubble observability.

## Karmada

- Official: <https://karmada.io/>
- GitHub: <https://github.com/karmada-io/karmada>

CNCF Incubating project providing multi-cluster and multi-cloud Kubernetes orchestration with a Kubernetes-native federation API.
