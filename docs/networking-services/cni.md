# CNI

## Cilium

- Official: <https://cilium.io/>
- GitHub: <https://github.com/cilium/cilium>

CNCF Graduated eBPF-based CNI providing networking, observability (Hubble), and security (network policies, mTLS) for Kubernetes.

## Calico

- Official: <https://www.tigera.io/project-calico/>
- GitHub: <https://github.com/projectcalico/calico>

Tigera's open source networking and network security solution for Kubernetes, supporting BGP, VXLAN, and eBPF data planes with rich NetworkPolicy.

## Multus CNI

- GitHub: <https://github.com/k8snetworkplumbingwg/multus-cni>

Kubernetes Network Plumbing WG meta-plugin that attaches multiple network interfaces to pods, enabling SR-IOV, MACVLAN, and secondary networks.

## Flannel

- GitHub: <https://github.com/flannel-io/flannel>

Simple, lightweight overlay network CNI for Kubernetes, providing layer-3 IPv4 networking with VXLAN, host-gw, or WireGuard backends.

## NSX Container Plugin (NCP)

- Official: <https://docs.vmware.com/en/VMware-NSX-T-Data-Center/index.html>

VMware's NSX-T Container Plugin that integrates Kubernetes networking and policies with VMware NSX for enterprise SDN environments.
