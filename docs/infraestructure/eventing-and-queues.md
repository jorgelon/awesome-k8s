# Eventing and Queues

## KEDA

- Official: <https://keda.sh/>
- GitHub: <https://github.com/kedacore/keda>

CNCF Graduated Kubernetes Event-Driven Autoscaler that scales workloads based on external event sources (queues, streams, databases, cron, etc.).

## Dapr

- Official: <https://dapr.io/>
- GitHub: <https://github.com/dapr/dapr>

CNCF Graduated distributed application runtime providing building blocks (pub/sub, state, bindings, workflows) via sidecars for any language.

## Argo Events

- Official: <https://argoproj.github.io/argo-events/>
- GitHub: <https://github.com/argoproj/argo-events>

CNCF Graduated event-driven workflow automation framework that triggers Kubernetes objects and Argo Workflows from many event sources.

## CloudEvents

- Official: <https://cloudevents.io/>
- GitHub: <https://github.com/cloudevents/spec>

CNCF Graduated specification for describing event data in a common, vendor-neutral way, with SDKs for many languages.

## NATS

- Official: <https://nats.io/>
- GitHub: <https://github.com/nats-io/nats-server>

CNCF Incubating high-performance messaging system with pub/sub, request/reply, and JetStream persistent streaming.

## Apache Kafka

- Official: <https://kafka.apache.org/>
- GitHub: <https://github.com/apache/kafka>

Distributed event streaming platform widely used for high-throughput pipelines, log aggregation, and event-driven architectures.

## CDEvents

- Official: <https://cdevents.dev/>
- GitHub: <https://github.com/cdevents/spec>

CDF specification for a common vocabulary of CI/CD events, enabling interoperability between continuous delivery tools using CloudEvents.

## Emitter

- Official: <https://emitter.io/>
- GitHub: <https://github.com/emitter-io/emitter>

High-performance, distributed pub/sub messaging broker built on MQTT, with security keys and clustering.

## Kubernetes Autoscaler (event-driven autoscaling components)

- GitHub: <https://github.com/kubernetes/autoscaler>

Kubernetes SIG repository containing Cluster Autoscaler, Vertical Pod Autoscaler, and the addon-resizer — components for elastic, event-driven scaling.

## Strimzi - Apache Kafka on Kubernetes

- Official: <https://strimzi.io/>
- GitHub: <https://github.com/strimzi/strimzi-kafka-operator>

CNCF Incubating operator for running and managing Apache Kafka clusters on Kubernetes, including Connect, MirrorMaker, and KRaft mode.
