# Serverless

## Knative

- Official: <https://knative.dev/>
- GitHub: <https://github.com/knative>

CNCF Graduated platform that adds serverless workloads (Serving and Eventing) to Kubernetes, with scale-to-zero and event-driven traffic routing.

## OpenFunction

- Official: <https://openfunction.dev/>
- GitHub: <https://github.com/OpenFunction/OpenFunction>

CNCF Sandbox FaaS platform on Kubernetes that combines Knative, KEDA, Dapr, and Shipwright to build, deliver, and run functions.

## OpenFaaS

- Official: <https://www.openfaas.com/>
- GitHub: <https://github.com/openfaas/faas>

Popular open source Functions-as-a-Service framework for Kubernetes, packaging any process or container as a serverless function.

## Apache Camel (K)

- Official: <https://camel.apache.org/camel-k/>
- GitHub: <https://github.com/apache/camel-k>

Camel K is a lightweight integration platform that runs Apache Camel integrations natively on Kubernetes as serverless workloads.

## Fission

- Official: <https://fission.io/>
- GitHub: <https://github.com/fission/fission>

Open source serverless framework for Kubernetes focused on fast cold-start functions in many languages.

## Nuclio

- Official: <https://nuclio.io/>
- GitHub: <https://github.com/nuclio/nuclio>

High-performance, data- and AI-oriented serverless platform with GPU support and real-time stream processing.

## Apache OpenWhisk

- Official: <https://openwhisk.apache.org/>
- GitHub: <https://github.com/apache/openwhisk>

Apache distributed serverless platform that executes functions in response to events at any scale, deployable on Kubernetes.

## Kogito Serverless Workflow

- Official: <https://kogito.kie.org/>
- GitHub: <https://github.com/apache/incubator-kie-kogito-serverless-operator>

Kogito's implementation of the CNCF Serverless Workflow specification, orchestrating cloud-native workflows on Kubernetes.
