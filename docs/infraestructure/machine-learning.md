# Machine Learning

## Kubeflow

- Official: <https://www.kubeflow.org/>
- GitHub: <https://github.com/kubeflow/kubeflow>

CNCF Incubating end-to-end ML toolkit for Kubernetes covering pipelines, notebooks, training operators, KServe, and Katib hyperparameter tuning.

## OpenSearch

- Official: <https://opensearch.org/>
- GitHub: <https://github.com/opensearch-project/OpenSearch>

Linux Foundation community-driven fork of Elasticsearch and Kibana, providing search, observability, and vector/ML capabilities.

## Elasticsearch

- Official: <https://www.elastic.co/elasticsearch>
- GitHub: <https://github.com/elastic/elasticsearch>

Elastic's distributed search and analytics engine, widely used for logs, observability, and vector search to back ML workloads.

## DataHub

- Official: <https://datahubproject.io/>
- GitHub: <https://github.com/datahub-project/datahub>

Open source metadata platform for the modern data stack, providing discovery, lineage, governance, and observability for data and ML assets.
