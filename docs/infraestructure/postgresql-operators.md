# PostgreSQL Operators

## PGO (Crunchy Postgres Operator)

- Official: <https://access.crunchydata.com/documentation/postgres-operator/latest/>
- GitHub: <https://github.com/CrunchyData/postgres-operator>

Crunchy Data's open source Postgres Operator for Kubernetes, providing production-grade HA, backups, monitoring, and disaster recovery.

## CloudNativePG

- Official: <https://cloudnative-pg.io/>
- GitHub: <https://github.com/cloudnative-pg/cloudnative-pg>

CNCF Sandbox Kubernetes operator (originated by EDB) for managing PostgreSQL clusters, with replication, failover, backups, and rolling upgrades.

## Postgres Operator (Zalando)

- GitHub: <https://github.com/zalando/postgres-operator>

Zalando's operator for managing PostgreSQL clusters on Kubernetes using Patroni, with logical backups and rolling upgrades.

## Percona Operator for PostgreSQL

- Official: <https://docs.percona.com/percona-operator-for-postgresql/>
- GitHub: <https://github.com/percona/percona-postgresql-operator>

Percona's open source Kubernetes operator for production-grade PostgreSQL clusters, with PMM monitoring, backups, and HA.

## StackGres

- Official: <https://stackgres.io/>
- GitHub: <https://github.com/ongres/stackgres>

OnGres' enterprise-grade PostgreSQL distribution for Kubernetes, bundling Patroni, PgBouncer, Envoy, Prometheus, and more out of the box.

## KubeDB

- Official: <https://kubedb.com/>
- GitHub: <https://github.com/kubedb/operator>

AppsCode's Kubernetes operator for many databases (PostgreSQL, MySQL, MongoDB, etc.), with backups, TLS, and lifecycle automation.

## Kubegres

- GitHub: <https://github.com/reactive-tech/kubegres>

Lightweight open source operator for deploying highly available PostgreSQL clusters on Kubernetes with streaming replication and failover.
