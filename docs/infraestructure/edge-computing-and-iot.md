# Edge Computing and IoT

## SuperEdge

- Official: <https://superedge.io/>
- GitHub: <https://github.com/superedge/superedge>

Open source edge container management system originally from Tencent, extending Kubernetes to edge locations with native distributed cluster support.

## KubeEdge

- Official: <https://kubeedge.io/>
- GitHub: <https://github.com/kubeedge/kubeedge>

CNCF Incubating project extending native Kubernetes orchestration to edge devices, with offline operation, device twin, and lightweight edge agent.

## Akri

- Official: <https://docs.akri.sh/>
- GitHub: <https://github.com/project-akri/akri>

CNCF Sandbox project for exposing heterogeneous leaf devices (IP cameras, USB, OPC UA, etc.) as Kubernetes resources at the edge.

## OpenYurt

- Official: <https://openyurt.io/>
- GitHub: <https://github.com/openyurtio/openyurt>

CNCF cloud-native edge computing platform extending upstream Kubernetes to manage workloads across cloud and edge nodes with autonomy on disconnection.

## Shifu

- Official: <https://shifu.dev/>
- GitHub: <https://github.com/Edgenesis/shifu>

Kubernetes-native IoT development framework that abstracts physical devices as deviceShifu microservices accessible over HTTP.

## FabEdge

- GitHub: <https://github.com/FabEdge/fabedge>

Secure edge computing networking solution built on Kubernetes and WireGuard, providing edge-to-edge and edge-to-cloud networking.
